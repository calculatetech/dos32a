# TASM Build Tool Media

This directory contains third-party historical tool media required by the
Windows DOS/32A build wrappers:

- Borland Turbo Assembler and Tools setup files from the TASM 5.x family.
- TASM32 5.3 DOS/Win32 patch files, including files identifying PSG Labs as the
  patch author.

These files are build tools, not DOS/32A source code. They are not covered by
the top-level DOS/32A license. Keep redistribution and provenance notes with any
future updates to this directory.
